# <span style="color: #FF6B35;">Choice Functions V1.0.0</span>

<span style="color: #00D4AA;">A collection of useful Python functions for selecting, filtering, and analyzing data.</span>

---

## <span style="color: #FF6B35;">Commands</span>

- <span style="color: #FF85A1;">Enter `choicehelp()` to see the commands.</span>
- <span style="color: #A66CFF;">Warning! Enter `import choice` or `from choice import *`. Then, write `choice.` before commands, so they will get called. If your variant is `from choice import*`, you don't need to write `choice.` before commands.</span>

---

## <span style="color: #00D4AA;">Recommendation</span>

- <span style="color: #4DA6FF;">Use `from choice import *` to use commands. `import choice` is also a working variant, but I recommend `from choice import *`.</span>

---

## <span style="color: #FFD93D;">Installation</span>

- <span style="color: #6BCB77;">Start the `install.cmd` file to begin installation. It will install required packages — Build, Setuptools, and Wheel. Then — Choice-functions.</span>
